# Zero-Shot Financial News Classifier

This project classifies financial news headlines using a zero-shot learning approach with the BART model.

## Features

- Uses `facebook/bart-large-mnli` for zero-shot classification.
- Supports multi-label classification for financial topics.
- Easily configurable candidate labels.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Place your data in the `data/` directory.

## Usage

Run the main script:
```bash
python src/main.py
```

## File Structure

- `src/config.py`: Model and label configuration.
- `src/data_loader.py`: Loads and preprocesses data.
- `src/classifier.py`: Zero-shot classification logic.
- `src/scorer.py`: Evaluation metrics.
- `src/main.py`: Entry point for running the pipeline.
- `data/cnbc_headlines.csv`: Example dataset.
